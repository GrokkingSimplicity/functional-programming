# 함수형 도구 체이닝

- 배열에 동작 이름과 제품 이름인 인자를 넣어 동작을 완전한 데이터로 표현하면 함수형 도구를 체이닝하기 좋다.

- 함수형 도구는 여러 단계의 체인으로 조합 가능하고 복잡한 계산을 작고 명확한 단계로 표현할 수 있다.

- 체인의 다음 단계를 위해 새로운 데이터를 만들거나 기존 데이터를 인자로 사용해야한다면 최대한 암묵적인 정보를 명시적으로 표현하는 방법을 찾아야한다.

## 체이닝 팁

### 데이터 만들기

- 함수형 도구는 배열 전체를 다룰 때 잘 동작하고, 배열 일부에 대해 동작하느 ㄴ반복문이 있다면 배열 일부를 새로운 배열로 나눌 수 있다.

### 배열 전체 다루기

- map으로 모든 항묵을 변환하고 filter로 항목을 없애거나 유지한다. 그리고 reduce로 항목을 하나로 합친다.

### 작은 단계로 나누기

- 두개 이상의 단계로 만들면 단순해서 이해하기 더 쉬워진다.

### 조건문을 filter로 바꾸기

- 반복문 안에서 조건문은 항목을 건너뛰기 위해 사용하는데, 앞에서 filter를 사용해 걸러보자

### 유용한 훔수로 추출

- 위에 썼던 함수형 도구 이외에 도구가 더 있고 스스로 찾을 수 있다. 함수를 추출하고 좋은 이름을 붙여 사용하자

# 중첩된 데이터에 함수형 도구 사용하기

- 어떤 깊이로 중첩된 데이터에도 동작할 수 있는 코드를 만들기 위해 재귀를 사용한다.
- 보통 일반적인 반복문은 재귀보다 명확하지만 중첩된 데이터를 다룰때는 재귀가 더 쉽고 명확하다
- 재귀는 스스로 불렀던 곳이 어디인지 유지하기 위해 스택을 사용하고 중첩된 데이터 구조를 그대로 반영한다.
- 많은 키를 갖고 있느 깊이 중첩된 구조에 추상화 벽을 사용하면 알아야할것이 준다. 추상화 벽으로 깊이 중첩된 데이터 구조를 쉽게 다루자
